# LINE AI Chatbot

This project is a LINE AI chatbot with file upload functionality, built using Flask and deployed on Vercel.

## Features
1. Upload files via LINE bot or browser.
2. View uploaded files.
3. Delete uploaded files.
4. Downsize images before uploading.

## Installation
1. Clone the repository.
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python3 app.py
